# 🌱 PlantPatho Monitor

PlantPatho Monitor is an AI + IoT based Smart Plant Disease Monitoring System designed for real-time plant health analysis and automated disease response.

The system uses an ESP32-CAM to capture leaf images, sends them to a Flask backend for AI-based disease classification using a MobileNetV2 model, and automatically controls a relay-based spray mechanism depending on the prediction result.

A modern responsive dashboard provides real-time monitoring, prediction history, system logs, reports, and device control features.

## 🚀 Features

- ESP32-CAM based image capture
- AI-powered plant disease detection
- MobileNetV2 deep learning model
- Flask backend API
- Automated relay/spray control
- Modern responsive dashboard
- Real-time monitoring and logs
- Report analytics and statistics
- WiFi-enabled IoT communication
